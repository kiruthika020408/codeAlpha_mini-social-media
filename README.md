# Mini Connect

Mini Connect is a simple social media web application built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. Users can create profiles, share posts, interact through comments, like posts, and follow other users.

## Features

- User Registration and Login
- User Profiles
- Create, Edit, and Delete Posts
- Comment on Posts
- Like and Unlike Posts
- Follow and Unfollow Users
- Responsive User Interface
- Secure Backend API

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/MiniConnect.git
```

2. Navigate to the project folder

```bash
cd MiniConnect
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
node server.js
```

5. Open your browser and visit

```text
http://localhost:5000
```
